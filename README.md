<h1> FlowY- A centralized management system app that for managing projects, tasks, roles all in one place and efficiently! </h1>

Tech Stack: </br>
Frontend: React (Typescript) , React Router, DND Kit </br>
Backend: Node.js (Express) </br>
Database: PostgreSQL </br>


<h1>Features</h1>
> Invite Users – Add team members to projects </br>
> Interactive Whiteboard – Real-time collaborative whiteboard for brainstorming. </br>
> A simplified Kanban Board - drag and drop functionality for ease of use  </br>
> basic role-based access - limit people's feature based on roles (Admin, Leader, Member) </br>
